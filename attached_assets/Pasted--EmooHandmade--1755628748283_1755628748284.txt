أنت صانع مواقع محترف. ابنِ موقعًا كاملاً وجاهزًا للإطلاق لعلامة مصرية اسمها “EmooHandmade” متخصصة في:

مناديل الفرح المطرزة (أسماء العروسين/التاريخ/دعاء قصير)،

طارات تطريز مخصّصة (أسماء، ورود، توليفات ألوان)،

ملابس مواليد مطرزة (بودي/سليبرز/قبعات)،

الطباعة على الماجات (صور/أسماء/عبارات)،

خدمات خياطة وتفصيل بسيطة (تصليح/تعديل/تفصيل طلبات صغيرة).

1) المتطلبات التقنية

أنشئ موقع سريع الاستجابة (Responsive) يدعم RTL عربي افتراضيًا مع اختيار التبديل إلى الإنجليزية (EN).

استخدم Next.js + React + Tailwind CSS.

أنماط نظيفة، قابلة للتخصيص، نظام مكوّنات Design System.

Animation: Framer Motion للحركات.

أيقونات: lucide-react.

نماذج: React Hook Form + Zod للتحقق.

صور: استخدم WebP/AVIF، تحميل كسول، توليد صورة مصغّرة.

Performance: LCP < 2.5s، CLS < 0.1، ضغط CSS/JS، prefetch للروابط الداخلية.

Accessibility: تباين ألوان AA، ARIA labels، دعم لوحة المفاتيح، خيار Reduce Motion.

2) الهوية البصرية (Egyptian Chic)

لوحة ألوان:

Ivory #F7F3EA (كخلفية قماش/كتان فاتح)

Blush #EEC9C5 (لمسات ناعمة)

Nile Blue #2F5D62 (للعناوين/الأزرار الرئيسية)

Gold Thread #C6A662 (تفاصيل مطرزة)

Charcoal #2B2B2B (نص أساسي)

خطوط ويب عربية/لاتينية:

العناوين العربية: Reem Kufi أو Cairo (سميك للعناوين).

نصوص عربية: Tajawal أو Amiri للنصوص الطويلة.

الإنجليزية: Poppins أو Inter.

زخارف: خامة كتان ناعمة جدًا (opacity منخفض)، زخارف تطريز (غرزة ساتان/سلسلة) SVG رقيقة حول العناوين.

صور/موك أب: حافظ على إضاءة طبيعية، خلفيات خشب فاتح/قماش، إبراز التفاصيل.

3) ميزة واجهة “المنديل الذي ينسدل”

أنشئ Section Component مخصص اسمه HankySection يكون شكله مثل “منديل” مطوي من الأعلى. عند الضغط على عنوان السكشن:

يحدث Reveal Animation: المنديل “ينسدل/يتدلّى” بسلاسة إلى أسفل، مع ظل بسيط يوحي بالعمق.

استخدم clip-path أو SVG mask لصنع الحواف المتموجة، مع حركة “fall & settle” (مدة 500–700ms، easing: cubic-bezier(0.22, 1, 0.36, 1)).

عند الفتح: يظهر المحتوى الداخلي (شبكة بطاقات/نص/نموذج)، مع staggered fade/slide.

وفّر حالات: مغلق افتراضيًا على الموبايل، مفتوح افتراضيًا على الديسكتوب للسكاشن المهمة (Hero، الكاتالوج).

أضف زر صغير أعلى المنديل (pin/chevron) لتبديل الفتح/الإغلاق، مع تلميح: “اضغط لعرض التفاصيل”.

اجعل المكون قابلًا لإعادة الاستخدام بكل الصفحات.

CTA رئيسي: “ابدأ تخصيص طلبك” + CTA ثانوي: “شاهد المعرض”

الكتالوج (Catalog)

أقسام فرعية (كل قسم بنفس مكوّن المنديل):

مناديل الفرح: فلاتر (نوع القماش، لون الخيط، نص/دعاء، تاريخ، سعر).

طارات التطريز: أحجام (10–25سم)، ورد مجسّم/مسطح، ألوان الخيوط.

ملابس مواليد: المقاسات، خامة القماش، اسم/تاريخ ميلاد، باكات (طقم 3/5 قطع).

ماجات مطبوعة: نوع الماج (سيراميك/ملون الداخل)، صورة/اسم/عبارة.

بطاقة المنتج: صورة أساسية + 3 صور، سعر، زمن التنفيذ، شحن، زر تخصيص.

ميزة Live Preview مبسطة: عند إدخال الاسم/التاريخ/اختيار اللون، توليد معاينة نصية على الصورة (موك أب).

صفحة منتج (Product Detail)

معرض صور، وصف مفصل، جدول مواصفات (خامة/مقاس/تعليمات عناية)، مربع تخصيص (اسم/تاريخ/ألوان)، وقت التنفيذ التقريبي، تكلفة إضافية إن وُجدت.

منتجات متعلقة/مكملة (Upsell): إضافة طارة/ماج بنفس الثيم.

معاينة مباشرة، وCTA: “أضف للسلة” أو “اطلب واتساب”.

10) تعليمات تنفيذ شكل “المنديل”

استخدم SVG كخلفية للمكوّن مع حواف مموجة و”غرزة” ذهبية رفيعة حول الإطار.

حالة مطوية: ارتفاع صغير، يظهر العنوان/الوصف المختصر + أيقونة سهم.

عند الضغط:

تدوير خفيف للأيقونة، تمدد المنديل لأسفل، drop shadow ناعم، اهتزاز خفيف عند الاستقرار (overshoot لطيف).

اجعل حدود المنديل تتكيف مع حجم المحتوى داخله (Auto Height).

وفّر خيار Compact Mode للموبايل (يظهر عنوان فقط حتى الضغط).

إمكانية تغيير لون “الخيط” في أطراف المنديل حسب القسم (ذهبي/أزرق/وردي).

12) نماذج وحقول التخصيص (أمثلة)

الحقول تعتمد على نوع المنتج:

الاسم/الأسماء (عربي/لاتيني)، التاريخ (اختيار بتقويم)، لون الخيط (لوحة)، نوع القماش/المقاس (قائمة)، رسالة قصيرة (حتى 60 حرفًا)، رفع صورة (للمجات/الطارات).

التحقق: عدم ترك الاسم فارغًا، طول الرسالة، حجم الملف ≤ 5MB.

عرض تكلفة إضافية مباشرة عند اختيار تفاصيل معينة (خيوط معدنية/حجم أكبر).

الإرجاع: المنتجات المخصصة غير قابلة للإرجاع إلا لعيب صناعة.
